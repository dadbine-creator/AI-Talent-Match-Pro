# AI Talent Match Pro

**Status:** Pre-Seed Development | Targeting LinkedIn Recruiter System Connect (RSC) Integration

**Project Overview:**
AI Talent Match Pro is a proprietary, serverless AI scoring engine designed to drastically reduce time-to-hire for high-demand technical roles.

**Unique IP (The Core Value):**
Our solution moves beyond keyword matching by using multi-dimensional prompt engineering to assess candidates on:
1.  **Adaptability Score:** Based on the variety of technologies and industries in their history.
2.  **Skill Recency:** Exponentially higher weighting given to skills used in the last two years.
3.  **Career Focus Penalty:** A metric to reward candidates with consistent, intentional career growth toward the target role.

**Architecture:** Designed for seamless integration with Azure Functions and Cosmos DB.
